## 1. Array.prototype.last()
> The **last()** method returns the last element of the array, or undefined if the array is empty.
### Syntax
```js
var last = [1,2,3].last();
```
### Examples
#### Using last()
The following example uses **last()** to get the last element of array.

```js
[1,2,3].last(); // 3
["a"].last(); // "a"
[].last(); // undefined
```
